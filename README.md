## Build

This project requires [.NET Core SDK 1.0 Preview 3](https://github.com/dotnet/core/blob/master/release-notes/preview3-download.md).
Build with Visual Studio 2017 or `dotnet`.

```
$ cd src/WeatherSite
$ dotnet restore
$ dotnet build
$ dotnet run
```
